HTTP response status codes. See docs.

Quick Start

import { OK } from 'costatus';

console.log(OK); // 200

Installation

NPM:

npm install costatus

Yarn:

yarn add costatus

Usage

ES Modules

Namespace import:

import * as costatus from 'costatus';

Named import:

import { OK } from 'costatus';

CommonJS

Namespace require:

const costatus = require('costatus');

Named require:

const { OK } = require('costatus');
